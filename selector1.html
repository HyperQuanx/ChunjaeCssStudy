    1. 전체 선택자

    3. 타입(태그) 선택자
      • 태그명으로 선택자 지정
      • 형식 : 태그명 { /* CSS 코드 */ }
    4. 아이디 선택자
      • id 속성값(아이디명)으로 선택자 지정
      • 형식 : #id명 { /* CSS 코드 */ }
    5. 클래스 선택자
      • class 속성(클래스명)으로 선택자 지정
      • 형식 : .class명 { /* CSS 코드 */ }
    6. 기본 속성 선택자
    7. 문자열 속성 선택자
  2. 조합(연결) 선택자
    • 선택자와 선택자를 연결해 적용하여 대상을 제한
    • combination selector(조합 선택자)
  3. 그룹 선택자
    • 여러 선택자를 콤마(,)를 사용하여 그룹으로 묶어서 지정
    • 형식 : 선택자1, 선택자2,,,,,, 선택자n { /* CSS 코드 */ }
  4. 자식 선택자(child selector)
    1. 자식 요소에 스타일을 적용
    2. 선택자 범위를 자식 관계로 제한
    3. 두 요소 사이에 >(부등호)를 이용하여 부모와 자식을 구분
    4. 형식 : 부모 선택자 > 자식 선택자 { /* CSS 코드 */ }
  5. 하위 선택자(descendant selector)
    • 선택자 범위를 자식 및 자손 관계로 제한
    • 부모 요소에 포함된 모든 하위 요소에 스타일 적용
    • 형식 : 상위 선택자 하위 선택자
    • 형식 : 선택자1 선택자2 선택자3,,,,{ /* CSS 코드 */ }
  6. 인접 형제 선택자(adjacent selector)
    • 특정 태그와 가장 인접한 형제 관계 태그를 선택자로 지정
    • 같은 부모를 가진 형제 요소 중에서 첫 번째 동생 요소에만 적용
    • 요소1과 요소2는 같은 레벨이면서 요소1 이후 첫 번째 요소2에 적용
    • 구분자 : +
    • 형식 : 이전 선택자 + 대상 선택자 { /* CSS 코드 */ }
  7. 일반 형제 선택자(sibling selector)
    • 특정 태그와 형제 관계에 오는 모든 태그를 선택자로 지정
    • 형제 요소들에 스타일 적용
    • 인접 형제 선택자와 다르게 모든 형제 요소에 적용
    • 구분자 : ~
    • 형식 : 이전 선택자 ~ 대상 선택자 { /* CSS 코드 */ }
