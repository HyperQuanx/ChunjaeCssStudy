스타일 기본

스타일 적용 순위 : 캐스케이딩
!important > 인라인 스타일 > id 스타일 > 클래스 스타일 > 타입 스타일
나중에 작성한 스타일이 먼저 작성한 스타일을 덮어씌움
글자와 관련된 속성
font-family : 글꼴 종류 지정
font-size : 글자 크기 지정
font-style : 글자 스타일 지정
font-weight : 글자 굵기 지정
color : 글자 색 지정
text-decoration : 텍스트 밑줄이나 취소선 표시할지 여부 지정
text-transform : 텍스트 전체, 또는 첫 글자를 대문자로 표시
text-shadow : 텍스트 그림자 추가
letter-spacing : 글자 사이의 간격 지정
word-spacing : 단어 사이의 간격 지정
text-align : 텍스트 정렬 방법 지정
line-height : 줄 간격 조절
웹 색상 지정은 16진수 방법을 가장 많이 씀 16진수 > RGBA > 색상 이름
width : 박스 모델 너비, height : 박스 모델 높이, box-sizing : 박스 모델 크기 계산하는 기준 지정, box-shadow : 박스 모델에 그림자 효과 추가
border 관련 속성은 대부분 한번에 적으니 안 나누고 한번에 첨삭 radius 는 둥글게, color는 색상 지정
margin, padding도 스킵.
display : 화면에 요소 배치 방법 지정. 자주 사용하는 주요 속성으로는 block, inline, inline-block, none, flex 등
position : relative, absolute, fixed 중요하니 layout.html, positionUnderstand.html 참조
위치는 top, bottom, left, right로 변경
background 관련 속성과 태그는 bg1~bg7 참조
