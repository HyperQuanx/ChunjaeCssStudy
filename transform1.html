변형(transform)

    • 변형(transform) : 특정 요소의 크기나 형태, 스타일 등이 바뀌는 것
    • 2차원/3차원 변형
      1. 수평/수직으로 웹 요소를 변형
      2. 크기/각도만 지정
      3. 2차원 좌표 사용
      4. 확대/축소
        1. scale(x,y) - 요소를 x, y 만큼 확대/축소
        2. scaleX(n) - 요소를 n만큼 x축으로 확대/축소
        3. scaleY(n) - 요소를 n만큼 y축으로 확대/축소
        4. scale3d(x,y,z) - 요소를 x, y, z축으로 각각 x, y, z 만큼 확대
      5. 기울이기
        1. skew(xdeg, ydeg) : 요소를 x축, y축으로 xdeg, ydeg만큼 기울임
        2. skewX(xdeg) : 요소를 x축으로 xdeg만큼 기울임
        3. skewY(ydeg) : 요소를 y축으로 ydeg만큼 기울임
      6. 회전
        1. rotate(deg) : 요소를 deg만큼 회전
      7. translate 함수
        1. 지정한 방향으로 이동할 거리 지정 후 이동
          • transform:translate(tx,ty) - x축 방향으로 tx만큼, y축 방향으로 ty만큼 이동, ty 값을 주지 않으면 0으로 간주
          • translate3d(tx, ty, tz)
          • translateX(tx) : x축 방향
          • translateY(ty) : y축 방향
          • translateZ(tz) : z축 방향
